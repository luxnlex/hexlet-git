Перед тем, как приступить к выполнению практического задания, выполните команду make reset

Если вы сделали что-то не так, и не знаете, как это исправить - вы можете выполнить команду make reset. В результате директория вернется в начальное состояние.

Файл README необходимо удалить
Файл lib/version.rb должен располагаться в директории lib/persey/version.rb
Вам необходимо привести репозиторий в порядок удалив файл README и переместив файл version.rb в нужное место.

Как всегда, вы можете решать эту задачу как вам удобно, но предпочтительно использовать команды git

Для каждого изменения (каждого файла) требуется создать по отдельному коммиту.

После того, как вы выполните 2 требуемых коммита, отправьте решение на проверку.
