Перед тем, как приступить к выполнению практического задания, выполните команду make reset

Если вы сделали что-то не так, и не знаете, как это исправить - вы можете выполнить команду make reset. В результате директория вернется в начальное состояние.

У вас есть проект, который размещается в директории project

Вам необходимо создать несколько файлов с выводом команды git log. Вручную создавать ничего не требуется, для создания файлов достаточно перенаправить вывод в файл. Это делается следующим образом: command > file, например: git log > log_1

В файл log_1 вывести всю историю коммитов, без применения каких-либо опций
В файл log_2 вывести историю последних 3-х коммитов, с отображением выполненных изменений в них
В файл log_3 вывести историю коммитов с ASCII-графом, показывающим историю ветвлений и слияний
В файл log_4 вывести историю коммитов в следующем формате: Сокращенный_sha - Имя_автора, Дата_автора : Комментарий, например, для последнего комита, это будет выглядить так: 1e9af36 - Hexlet git user, Sat Mar 21 18:06:15 2015 +0300 : Ignore .sw* files
Каждый файл нужно будет поотдельности проиндексировать и выполнить коммит (1 коммит с 1 файлом). Выполнять коммиты нужно ПОСЛЕ того, как вы сгенерируете все 4 файла.

После того, как вы выполните все 4 коммита с файлами, отправьте решение на проверку.
